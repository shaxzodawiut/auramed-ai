# AuraMed AI

AI-based contactless system for surgeons.

## Problem
During surgery, doctors need to interact with medical imaging (CT/MRI) but cannot touch devices without breaking sterility or relying on assistants, which causes delays.

## Solution
AuraMed AI enables surgeons to control medical imaging using hand gestures via computer vision, without any physical contact.

## Key Features
- Gesture-based navigation (scroll, zoom, rotate)
- No additional hardware (standard webcam)
- Real-time interaction
- Designed for operating room conditions
